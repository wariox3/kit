Editor de listas de precios
El editor de listas de precios es una herramienta diseñada para manipular los datos de las listas de precios. Estas listas tendrán la información de los precios con los cuales funcionara toda clase de liquidacion que intervenga en el proceso de logística.

Esta herramienta posee funciones de edición en rama de archivos de precios, para facilitar la edición de estas base de datos. 

En estos libros de ayuda encontrará toda la información necesaria para, crear, editar, eliminar y manipular tanto las plantillas de precios como los datos que estas contengan, con trucos de edición que le ahorraran mucho tiempo a la hora de trabajar.
Cabe aclarar que las listas de precios pueden estar contenidas en la base de datos o en archivos planos, los cuales pueden ser transportados con facilidad y editados con el programa.
