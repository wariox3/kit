Formato
El formato de estas plantillas de precios es *.mdb, *.elp los cuales son archivos nativos del editor de bases de datos de Microsoft "Microsoft Acces", con una estructura de tablas y consultas básicas para la relación objetiva y almacenamiento de la información. Cabe la aclaración de que no es necesario tener instalado Acces en el sistema para trabajar con estas listas de precios porque el editor utiliza el Microsoft Data access components, el cual es gratuito y nos ofrece algunas herramientas para acceder a este tipo de archivos de datos.

  • Tablas:    Ciudades, Precios, Productos
  • Consultas: Informe Precios

Nota: No se recomienda que los usuarios modifiquen las plantillas desde otro editor de estos archivos diferente a Que!elp, pues, la información puede dañarse o no arrojar los resultados esperados, ya que los datos dependen unos de otros y la eliminación de un registro por ejemplo de ciudades afecta todos los precios de esa ciudad. 

Que!software no se hace responsable por ningún daño o falla ocasionada por la manipulación en plano de estas plantillas de datos.
