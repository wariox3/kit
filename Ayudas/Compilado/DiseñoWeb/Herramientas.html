Introducción a las herramientas adicionales
­
K!elp [Editor de listas de precios]
K!elp es el editor por defecto de las plantillas o archivos de datos, con la información de los precios con los cuales funcionara toda clase de liquidaciones que intervengan en el proceso de logística.

Esta herramienta posee funciones de edición en rama de archivos de precios, para facilitar la edición de estas base de datos. 

En estos libros de ayuda encontrará toda la información necesaria para, crear, editar, eliminar y manipular tanto las plantillas de precios como los datos que estas contengan, con trucos de edición que le ahorraran mucho tiempo a la hora de trabajar.

K!eci [Editor de coordenadas de impresión]
K!eci es el editor de coordenadas de impresión, que le ayudara a ajustar las posiciones donde desee que se impriman los datos para los archivos mas comunes
