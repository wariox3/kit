Presentación
En este recuadro aparece el titulo de la aplicación, el sistema comprueba que existan los archivos de información, las bases de datos, que los servidores de la base de datos estén ejecutados etc. Además aquí el sistema registra el usuario licenciado del software, la empresa a la que pertenece y su clave o serial.

1- Que!software, es la empresa que elaboro, edita y actualiza el software, para mayor información entre en el siguiente link http://200.75.74.93/kit_logistics/index.asp

2- Este es el nombre del modulo o aplicativo al que vamos ingresar, además, lleva consignada la versión en que se encuentra.

3- Nombre del paquete al que pertenece este aplicativo.

4- Nombre de la persona que compro la licencia del programa.

5- Empresa para la cual esta habilitada la licencia.

6- Serial, con el cual se va a reconocer individualmente cada producto.

Nota: Si el software no pasa de esta pantalla por favor llame a soporte tecnico.
