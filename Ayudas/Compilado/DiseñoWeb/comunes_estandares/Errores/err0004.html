Error 0004
­ ­
Especificaciones del error
­
 

Propiedades, descripción y solución

 Descripción Solución
Este error ocurre cuando el usuario intenta imprimir un anuncio que ya esta impreso.
Verifique que el anuncio no este impreso, si ya se imprimió el anuncio y necesita volver a imprimirlo, en el menú edición existe una opción que se llama [Re-imprimir anuncio] esta vuelve a sacar una orden de recogida (impresión de anuncio). Cabe anotar que debe tener el permiso especial para esta acción.
