Propiedades, descripción y solución

 Descripción Solución
Este error ocurre porque el sistema tiene del configurada la ruta de acceso de al archivo de configuraciones del modulo de recogidas. Ingrese al aplicativo administrador (solo lo puede hacer el administrador que para este caso es el único que tiene una contraseña especial para realizar este tipo de cambios)
- Menú herramientas
- Configurar archivos de información
- Seleccione el ítem "Recogidas", en la columna "existe" debe decir "NO"
- Presione el botón Cambiar ruta y seleccione el archivos de extensión .kin llamado [infrecogidas].
- Presione abrir
- Clic en aceptar
