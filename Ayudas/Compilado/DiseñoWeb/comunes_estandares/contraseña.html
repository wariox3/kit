Contraseña... un anillo de seguridad al alcance del usuario.

En este formulario se selecciona el usuario del combo con el titulo; se escribe la contraseña, luego se presiona en aceptar y en el caso de que los datos sean correctos, el sistema accede a la consola o pantalla principal.
De este anillo de protección, el sistema carga unos permisos para las acciones en cada uno de los módulos de operación, así, un usuario podrá entrar a un modulo a crear y modificar registros pero no podrá eliminar. Cada uno de los formularios tendra sus respectivos permisos, para ingresar, crear, editar y eliminar o anular. 


Nota: En caso de no aparecer en el combo de usuarios consulte con el administrador porque puede ser que no este creado o lo hayan creado con otro nombre.

Para administrar los permisos de un usuario se accede por el modulo de Administrador.
