Pasos para ingresar un anuncio de recogida
­
 

Primero debe ingresar al modulo de ingreso de anuncios, en el menú de mantenimiento clic en la opción [Ingresar anuncios], si sus permisos están habilitados saldrá esta información en un formulario.

1- Cliente: En este campo se ingresa el documento con el cual fue creado el cliente, para el caso de clientes con negociación y datos adicionales, en el momento de ingresar el nit y presionar la tecla entrar, el sistema automáticamente le trae la información para los campos de anunciante, nombre, dirección de bodega, teléfono, ruta que ya están parametrizados en la información del cliente si el cliente es esporádico se digita 0 o sencillamente no se coloca nada en este campo de código y el sistema asume que el cliente es "Varios".

2- Información de la recogida: En esta sección del formulario, se ingresa el horario y la fecha en la que se debe recoger la mercancía, además, la ruta urbana a la que se le debe asignar esta recogida en el momento de la programación. Debemos introducir las unidades y el peso de la mercancía que se va a recoger. Si la operación lo requiere puede colocar notas adicionales en el anuncio.

3- Información de programación: Estos campos no son aditables manualmente, se va actualizando con las operaciones del proceso y contienen la información del estado de la recogida y dato del momento en que se recogió la mercancía.

4- Asignación: Aquí podemos ver que vehículo fue programado para recoger, cuanto se le pago y ver otras recogidas programadas para este mismo recorrido.
